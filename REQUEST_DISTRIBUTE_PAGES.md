# Request & Distribute - 2 Page System

## Page 1: Request Food (request.html)

### Purpose
Browse and request available food donations from donors

### Sample Items to Search:
1. **Vegetable Biryani** - 50 servings, New York
2. **Fresh Vegetables** - 30 servings, Brooklyn

6. **Fruit Salad** - 20 servings, New York
7. **Pasta Primavera** - 35 servings, Brooklyn
8. **Fish Fry** - 30 servings, Queens

### Features:
- Search by food name, location, or donor name
- Filter by Vegetarian/Non-Vegetarian
- View food details (quantity, expiry, location)
- Request food button for each item
- Real food images

### Search Examples:
- Try: "Biryani", "Vegetables", "Rice", "Chicken"
- Try: "New York", "Brooklyn", "Manhattan", "Queens"
- Try: "Green Restaurant", "Farm Fresh", "Community Kitchen"

1. **Request Page** → Browse available food → Click "Request Food"
2. **Distribute Page** → View accepted requests → Coordinate pickup → Mark as picked up → Distribute to beneficiaries

## Navigation:
