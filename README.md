# FoodBridge - Food Waste Reduction Platform

## Overview
A web application connecting food donors with NGOs to reduce food waste and improve food security.

## Features
- User registration and authentication (Donor/NGO roles)
- Food donation management with image upload
- Browse and search available donations
- Request food functionality for NGOs
- Role-based dashboards with statistics
- Real-time filtering and search

## Technologies
- HTML5, CSS3, JavaScript
- LocalStorage for data persistence
- Responsive design

## Images Used
All images are from Unsplash (free to use):

- How it works: Food preparation, search, delivery
- Donate page: Food sharing
- Browse page: Fresh vegetables
- Dashboard: Statistics/analytics

## How to Run
1. Open `index.html` in a web browser
2. Register as Donor or NGO
3. Login and start using the platform
