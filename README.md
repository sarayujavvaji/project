# FoodBridge - Food Waste Reduction Platform
## FSAD-PS09: Reduce Food Wastage and Improve Food Security

## Overview
A comprehensive platform to track and reduce food waste. The app offers solutions to manage food resources more efficiently, help people understand the impact of food waste, and improve overall food security.

## User Roles

### 1. Admin
- Manage platform content and oversee user interactions
- Monitor all donations and requests
- View platform statistics and user management
- Ensure data accuracy across the platform

### 2. Food Donor
- List surplus food with details and images
- Coordinate donations with recipient organizations
- Track donation impact and history
- View donation statistics and completion rates

### 3. Recipient Organization
- Request food donations based on needs
- Manage logistics for food collection
- Distribute food to those in need
- Track received donations and servings

### 4. Data Analyst
- Track food waste trends and patterns
- Analyze donation data and generate insights
- View food type distribution and metrics
- Generate reports to improve platform efficiency

## Features
- Multi-role user registration and authentication
- Food donation management with image upload
- Browse and search available donations
- Request food functionality for recipient organizations
- Role-based dashboards with specific statistics
- Real-time filtering and search
- Data analytics and trend tracking
- Admin oversight and user management

## Technologies
- HTML5, CSS3, JavaScript
- LocalStorage for data persistence
- Responsive design

## How to Run
1. Open `index.html` in a web browser
2. Register as Admin, Food Donor, Recipient Organization, or Data Analyst
3. Login with your credentials
4. Access role-specific dashboard and features

## Role-Specific Access
- **Admin**: Full platform oversight, user management, and statistics
- **Food Donor**: Donate food page, track donations, view impact
- **Recipient Organization**: Browse donations, request food, manage logistics
- **Data Analyst**: View analytics, trends, reports, and insights
